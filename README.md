## Taskbar 2

A standalone taskbar bar widget ported from Noctalia Shell, with improvements and fixes:

  - Fixes unresponsiveness with rapidly updating window titles.
  - Adds per-app grouping, optionally for each workspace.
  - Adds workspace separators with prefix/suffix.
  - Adds title and icon styling settings (colors, states, scale)
