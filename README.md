# rigerc's noctalia-plugins

> 🤖 This is a vibe coded project built by Codex and me. Treat the code and behavior as experimental, and verify changes before relying on it.

## Plugins
- **[CodexBar](codexbar/README.md)** — AI provider usage stats from the codexbar CLI in your bar.
- **[Scrollbar](scrollbar/README.md)** — A focused, scrolling window strip for the bar with centered selection and customizable overflow edge cues.
- **[Scrollbar2](scrollbar2/README.md)** — A configurable window bar with equal-width segments, animated focus tracking, and optional floating panel placement.
